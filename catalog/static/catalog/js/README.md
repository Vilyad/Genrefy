# JavaScript для Genrefy

## Функциональность

### Основные возможности:
1. **Анимация карточек** - плавные эффекты при наведении
2. **Избранное** - сохранение состояния в localStorage
3. **Toast-уведомления** - всплывающие сообщения
4. **Адаптивность** - оптимизация для мобильных устройств
5. **Форматирование чисел** - автоматическое (1K, 1M)
6. **Аналитика кликов** - логирование действий пользователя

### Использование:
```javascript
// Доступ к приложению через window.Genrefy
window.Genrefy.showToast('Привет!');
```
